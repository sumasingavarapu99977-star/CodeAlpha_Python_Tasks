# Python Internship Projects

## Overview

This repository contains Python mini projects developed during my internship. These projects helped me improve my programming skills, problem-solving ability, and understanding of Python fundamentals such as loops, conditions, functions, dictionaries, file handling, and user interaction.

The projects included in this repository are:

1. Chatbot Application
2. Hangman Game
3. Stock Portfolio Tracker

# 1. Chatbot Application

## Project Description

The Chatbot Application is a simple text-based chatbot developed using Python. The chatbot interacts with the user by responding to predefined messages and continues the conversation until the user exits the program.

This project demonstrates the basic implementation of conversational programs using conditional statements and loops.

---

## Objectives

* To understand user interaction in Python
* To implement loop-based conversation flow
* To learn conditional statements
* To improve logical thinking

---

## Features

* Greets the user
* Responds to simple questions
* Supports continuous conversation
* Exit option using "bye"
* User-friendly interface

---

## Technologies Used

* Python
* Jupyter Notebook

---

## Concepts Used

* while loop
* if-elif conditions
* input and output functions
* string handling

---

# 2. Hangman Game

## Project Description

Hangman is a word guessing game developed using Python. The computer selects a random word, and the player tries to guess the word one letter at a time before running out of chances.

This project demonstrates the use of loops, lists, strings, and the random module.

---

## Objectives

* To build a console-based game
* To understand game logic
* To improve problem-solving skills
* To practice loops and conditions

---

## Features

* Random word generation
* Letter guessing system
* Displays correct guesses
* Limited number of attempts
* Win and lose conditions

---

## Technologies Used

* Python
* Random module

---

## Concepts Used

* Conditional statements
* String manipulation
* Random module

# 3. Stock Portfolio Tracker

## Project Description

The Stock Portfolio Tracker is a Python-based application used to calculate and manage stock investments. Users can enter stock names and quantities, and the application calculates the investment value and total portfolio value.

The project also stores the portfolio report in a text file.

---

## Objectives

* To manage stock investment data
* To perform calculations using Python
* To implement file handling
* To build a menu-driven application

---

## Features

* Displays available stocks
* Accepts user stock selection
* Calculates investment value
* Computes total portfolio value
* Saves portfolio report to file
* Handles invalid stock names

---

## Technologies Used

* Python
* VS Code

---

## Concepts Used

* Dictionary
* Functions
* Loops
* File handling
* Exception handling
